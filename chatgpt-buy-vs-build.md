# Buy vs Build Analysis
**Platform:** ChatGPT (GPT-4 / GPT-5)

## System Message
You are a strategic analyst specializing in build-versus-buy decisions for technology products. You provide comprehensive analysis considering technical feasibility, financial impact, strategic value, and risk.

## User Message
Conduct a buy-vs-build analysis for a product capability.

**Capability:** [Feature/system needed]
**Context:** [Product stage, team size, urgency]
**Budget:** [Available resources]

### Analysis Required:

**Build Analysis:**
- Development cost and timeline
- Ongoing maintenance burden
- Technical risks and mitigation
- Team capability gaps
- Strategic value of ownership

**Buy Analysis:**
- Vendor options comparison table
- Integration effort and risk
- Total cost of ownership (3-5 years)
- Vendor dependency assessment
- Customization limitations

**Hybrid Options:**
- Build core + buy components
- Partner with customization rights
- Acquire team/startup

**Decision Framework:**
Create scoring matrix across dimensions:
- Strategic alignment (1-10)
- Time to market (1-10)
- Cost efficiency (1-10)
- Risk level (1-10, lower better)
- Differentiation value (1-10)

**Recommendation:** Clear choice with supporting evidence and sensitivity analysis.
